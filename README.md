# Official web site for iBitly.com

## Add or update posts 

- Add or update post at folder: _posts.
- Test local with: npm run dev
- scp -r out/*  root@47.95.28.64:/usr/share/nginx/html/test_blog/.

## Publish website 

-- Run: npm run build
-- Push to web server: scp -r out/*  username@host:/html.
